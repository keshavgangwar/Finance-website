Optional Fields: Generate Unique Loan Numbers
The below can be used to generate a unique Loan # in a certain format on the Add Loan page. By default, the system generates a unique 7 digit Loan #. But you might want a different format or a branch code in that number. You can use the below placeholders to force the system into generating loan numbers based on the below pattern.
Examples
  • To generate BR-10001, BR-10002 and so on, you would type BR-{U10001} below.
  • To generate LR-Year-Month-10001, LR-Year-Month-10002 and so on, you would type LR-{yyyy}-{mm}-{U10001} below.
  • To generate 10001-YearMonthDay, 10002-YearMonthDay and so on, you would type {U10001}-{yyyymmdd} or {U10001}-{yymmdd} below.
  • To generate LRO-10-BR-11111, LRO-10-BR-11112 and so on, you would type LRO-10-BR-{U11111} below.
  • To generate LRO10BR11111, LRO10BR11112 and so on, you would type LRO10BR{U11111} below.
